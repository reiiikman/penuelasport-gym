# PeñuelaSport GYM — Landing

Landing de captación para **PeñuelaSport GYM** (Coquimbo, Chile).
HTML/CSS estático, autocontenido, sin dependencias de red.

- `index.html` — la página (una sola sección scroll).
- `pretext.js` — runtime de cómputo de texto ([Pretext](https://esm.sh/@chenglou/pretext)), local.

## Ver localmente
```
python -m http.server 5500
```
Abre http://localhost:5500

## Datos
Planes, horario, entrenadores, nutrición y contacto reales del gimnasio.
